# Brealu Outdoor - Premium Bioclimatic Solutions

Official website for **Brealu Outdoor**, a global leader in premium aluminum pergolas, carports, and architectural shading solutions.

## Brand Essence
- **Slogan**: Breathe · Relax · Embrace.
- **Mission**: Elevating outdoor living with precision-engineered bioclimatic systems.
- **Differentiator**: 24-hour custom 3D design service for complex architectural footprints (L-shape, U-shape, etc.).

## Project Structure
- `index.html`: Main landing page featuring a modern DTC layout, project showcase, factory strength, and inquiry forms.
- `product-marketing-context.md`: Comprehensive brand positioning and marketing strategy.
- `site-blueprint.md`: Sitemap and architectural planning for future expansions.

## Deployment Information
This site is designed to be hosted via **Cloudflare Pages** for high performance and global availability.

### How to Deploy
1. Push this repository to GitHub.
2. Connect the repository to a new project in the [Cloudflare Pages Dashboard](https://dash.cloudflare.com/).
3. Set the build command to `(None)` and the output directory to `.` (root).
